**가장 빠르게 소설 작성기를 시작하는 방법!**

---

## ⚡ 가장 빠른 방법

### 추천: 로컬 서버 (3초)
```
1. START-SERVER.bat 더블클릭
2. 브라우저 자동 실행
3. 끝!
```

→ `http://localhost:3000` 모든 기능 작동 ✅

---

### 대안: 구 버전 파일 (0초)
```
1. novel_writer_v3_step3.html 더블클릭
2. 끝!
```

→ 최신 기능은 없지만 파일로 바로 실행 가능

⚠️ **index.html은 파일로 직접 열기 불가**
- 최신 브라우저가 보안상 ES6 모듈을 차단합니다
- 반드시 로컬 서버로 실행해야 합니다

---

## 📝 기본 사용법

### 1. 프로젝트 시작
- 왼쪽 상단 "프로젝트명" 클릭
- 제목, 장르, 줄거리 입력
- 자동 저장됨!

### 2. 회차 작성
- 왼쪽 패널에서 "회차 추가" 클릭
- 제목 입력 → 본문 작성
- `Ctrl+S` 또는 자동 저장

### 3. Google Drive 연동 (선택)
- 상단 "Google Drive 연동" 버튼 클릭
- 로그인 → 권한 승인
- 자동 백업 시작!

### 4. Claude AI 검토 (선택)
- "검토" 탭 클릭
- API 키 입력 (`sk-ant-api-...`)
- 본문에 텍스트 입력
- "검토 요청" 클릭
- AI 피드백 받기!

## 💡 자주 묻는 질문

### Q: 설정이 복잡한가요?
A: **아니요!** 파일 더블클릭만 하면 바로 사용 가능합니다.

### Q: Google Drive 연동 필수인가요?
A: **아니요!** 로컬 저장만 써도 됩니다.
   - 자동 저장: localStorage (브라우저)
   - 수동 백업: "다운로드" 버튼 (JSON 파일)

### Q: Claude API 없이도 되나요?
A: **네!** 기본 글쓰기 기능은 모두 작동합니다.
   - AI 검토 기능만 사용 불가
   - 캐릭터 자동 생성만 사용 불가

### Q: 인터넷 연결 필요한가요?
A: **로컬 사용 시 불필요**
   - Google Drive: 인터넷 필요
   - Claude API: 인터넷 필요
   - 기본 글쓰기: 오프라인 가능

### Q: 다른 컴퓨터에서도 쓸 수 있나요?
A: **네!** 3가지 방법:
   1. 파일 폴더 복사 → 다른 PC에서 실행
   2. Google Drive로 동기화
   3. GitHub Pages로 배포 (어디서나 접속)

### Q: 스마트폰에서도 되나요?
A: **부분적으로 가능**
   - GitHub Pages로 배포하면 모바일 접속 가능
   - 로컬 파일은 PC 전용
